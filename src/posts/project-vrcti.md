2026년 4월 2일, VRChat 유저들을 위해 맞춤형으로 제작된 인디케이터 시스템인 VRCTI 배포를 완료했어요! 🎉

## 기술 개요 (Technical Overview)
이 프로젝트는 커스텀 VRChat 지표들을 시각적으로 보여주기 위해 디자인된 프론트엔드 웹 애플리케이션이에요:
- **E/I**: 외향성(Extraversion) / 내향성(Introversion) 상태.
- **S/C**: 소셜(Social) / 칠(Chill) 상태.
- **D/G**: 데스크탑(Desktop) / VR 고글(VR Goggles) 모드.
- **A/U**: 자리 비움(Away) / 방해 금지(Unavailable) 상태.

## 배포 및 호스팅 (Deployment & Hosting)
이 애플리케이션은 100% 정적(Static)으로 생성되었고 GitHub Pages를 통해 호스팅되고 있어요. 여기에서 실제 배포된 사이트를 확인해 볼 수 있답니다: [VRCTI Live](https://nupamo.github.io/vrcti/)

정적 호스팅을 활용하면 스케일링 비용이 전혀 들지 않고 엣지(edge) 전송이 빨라서, 가벼운 인디케이터 대시보드를 위한 최적의 기술 스택 선택이라고 할 수 있어요! ✨
