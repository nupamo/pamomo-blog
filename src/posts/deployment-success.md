## 🎉 배포 완료!

기존 `dev-blog`라는 이름에서 **파모모의 블로그(Pamomo's Blog)**로 새단장을 마치고, GitHub Pages에 무사히 배포되었습니다.

### 주요 변경 사항
- 프로젝트 이름 및 UI 브랜드명 변경 ('파모모의 블로그')
- `vite.config.ts`의 `base` 경로를 `/pamomo-blog/`로 설정
- `gh-pages` 패키지를 이용한 자동 배포 스크립트 구성

라이브 URL: [https://nupamo.github.io/pamomo-blog/](https://nupamo.github.io/pamomo-blog/)
