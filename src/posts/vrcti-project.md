## VRCTI: VRChat 인디케이터 시스템 구축

VRChat 유저들을 위한 맞춤형 상태 인디케이터 웹 애플리케이션(VRCTI)을 성공적으로 배포했어요! 🎉
- **기능**: E/I(외향/내향), S/C(소셜/칠), D/G(데스크탑/VR), A/U(자리비움/방해금지) 상태 표시
- **배포**: GitHub Pages 정적 호스팅을 활용해 유지보수 비용 없이 가볍게 구축했습니다.

## AI DevOps 비용 최적화와 트래킹

개발 환경에서 AI 사용량이 늘어남에 따라 API 토큰 운영 비용을 모니터링하고 최적화하는 작업도 진행했어요. 📊
- **비용 관리**: 대화 컨텍스트를 압축(`/compact`)하고 불필요한 프롬프트 주입을 줄여 API 토큰 소모를 최적화했습니다.
- **포트폴리오 트래킹**: `portfolio.json`을 도입해 프로젝트 성과를 체계적으로 추적하고 AI가 쉽게 참조할 수 있게 만들었어요!
